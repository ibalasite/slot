Greek Mythology Slot
Engineering Documentation

Thunder Blessing is a high-volatility Greek-mythology slot featuring Cascade expanding reels (up to 5×6), Lightning Mark chain elimination, and a ×77 Free Game multiplier unlocked via Coin Toss. This documentation suite covers the complete GDD pipeline from business requirements through production runbook — every spec is designed to be directly executable by engineering and QA.

Game Mechanics Summary

Mechanic Specification
Reel Layout 3×5 base → expands to 6×5 (up to 57 paylines) on winning Cascade
Symbol Set 4 Premium (P1–P4) + 4 Low (L1–L4) + Wild (W) + Scatter (SC) + Lightning (⚡)
Cascade Mechanic Winning symbols removed; new symbols fall; reel adds 1 row per cascade step
Lightning Marks Accumulated from cascade wins; Scatter detonation converts all marked cells to highest-payout symbol on grid
Coin Toss Triggered at Free Game entry; sequence ×3 → ×7 → ×17 → ×27 → ×77 selects FG multiplier
Free Game (FG) 10 spins minimum; Heads outcomes extend FG; Extra Bet guarantees Scatter on next FG spin
Extra Bet Cost ×3 baseBet; guarantees Scatter appearance in main grid
Buy Feature Cost ×100 baseBet; triggers Heads×5 FG sequence immediately; session floor ≥ 20× baseBet
RTP Scenarios Base 97.5% | ExtraBet 97.5% | BuyFG 97.5% | ExtraBet+BuyFG 97.5%
Max Win (Main) 30,000× baseBet
Max Win (Buy FG) 90,000× baseBet (Extra Bet + Buy FG combined)
Volatility Medium-High
Currency USD ($0.10–$10.00 baseBet) | TWD (TWD 3–60 baseBet)

Technology Stack

LayerTechnologyNotes
Frontend Cocos Creator / PixiJS Canvas renderer; responsive layout
Backend API TypeScript + Fastify Clean Architecture; single-trip spin endpoint
Game Engine slot-engine (internal) Excel-driven probability toolchain; verify.js gate
Database PostgreSQL (Supabase) Audit log + player sessions
Cache Redis Session state; rate limiting; spin lock (NX)
Auth Supabase Auth (RS256 JWT) 3600s access token; 7-day refresh
Infrastructure Kubernetes + nginx-ingress TLS 1.3; HSTS; circuit breakers
CDN Cloudflare / CloudFront Static assets; edge caching
Probability Source Excel + BetRangeConfig.generated.ts No manual param editing allowed
